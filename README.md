📌 Project Overview

This project is a rule-based AI chatbot designed to assist users in filing insurance claims for:

- 🚗 Vehicle Insurance
- 🏥 Health Insurance
- ❤️ Life Insurance

The chatbot provides step-by-step guidance using a JSON knowledge base.

---

🎯 Objectives

- Simplify insurance claim process
- Provide guided conversational support
- Implement chatbot using Flask backend
- Use rule-based NLP for intent detection

---

🏗️ Tech Stack

- Backend: Python Flask
- Frontend: HTML, CSS, JavaScript
- Data: JSON Knowledge Base
- NLP: Keyword-based matching

---

⚙️ Features

- 📋 Step-by-step claim guidance
- 📑 Document checklist
- 📊 Claim status simulation
- 💬 Interactive chat UI
- ⚡ Fast response (no external API)

---

🔄 Workflow

1. User sends message
2. Chatbot detects intent using keywords
3. Retrieves data from JSON
4. Sends response to UI

---

▶️ How to Run

pip install flask
python app.py

Open browser:

http://127.0.0.1:5000

---

⚠️ Limitations

- Rule-based (not true AI)
- Limited to predefined queries
- No real-time database
